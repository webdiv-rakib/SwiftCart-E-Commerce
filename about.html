About SwiftCart

SwiftCart is a modern, fast, and user-friendly online shopping platform created as a learning project. We aim to provide the best experience for both customers and developers exploring e-commerce web development.

Our Mission

To build clean, responsive, and real-data-driven e-commerce sites using only HTML, CSS (Tailwind + DaisyUI), and vanilla JavaScript.

What We Offer

  • Real-time product data from Fake Store API
  • Category filtering & search-like browsing
  • Cart system with localStorage persistence
  • Fully mobile-responsive design
  • Beautiful UI powered by DaisyUI themes

Fun Fact

This entire site was built step-by-step with real API calls.
